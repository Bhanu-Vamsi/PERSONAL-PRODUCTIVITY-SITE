Personal Productivity Suite
===========================
A simple Python desktop application built with Tkinter containing:
- Calculator: safe arithmetic expression evaluator
- Notes: create, save and load notes
- Timer: simple countdown timer with start/stop
- File Organizer: organize files in a selected folder by extension

How to run
----------
1. Make sure you have Python 3.8+ installed.
2. Run: python main.py
No external packages required (uses standard library).
